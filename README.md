# Steam Workshops Tools 创意工坊下载工具

这是一款下载 Steam 创意工坊模组的工具，支持所有可通过匿名账户下载的模组，并支持部分需要正版账户的模组。

在 [Releases](https://github.com/King-zzk/Steam-Workshops-Tools-SWTools/releases) 下载最新版本即可使用。
